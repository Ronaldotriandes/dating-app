## Installation

Use the package manager [npm](https://www.npmjs.com/) to install 
Or use with yarn to install 

```bash
yarn install --save
```
## Set ENV
```bash
PORT = 
DB_TYPE = postgres
DB_PORT = 5432
DB_HOST = 
DB_NAME = 
DB_USERNAME = 
DB_PASS = 
JWT_SECRET = 
JWT_EXPIRES_IN =
```

## Run this app with yarn
```bash
yarn run start:dev
```

## Run a test
```bash
yarn run start
```
